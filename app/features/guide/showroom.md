# たかねこの SHOWROOM 配信

SHOWROOM はライブ配信サービスで、多数のアイドルが配信を行っています。

高嶺のなでしこのメンバーは主に SHOWROOM を利用して、不定期に配信を行っています。
メンバーの配信では Instagram Live や TikTok ライブではなく、SHOWROOM が利用されています。

## 配信について

配信タイミングは不定期です。

自分の誕生日には誕生日配信が行われます。
ぜひ参加してお祝いしましょう。

配信の内容はメンバーによってまちまちです。

- ライブの意気込み・振り返り
- 動画の撮影時の裏話
- 雑談・コメント読み
- カラオケ
- 誕生日配信

## ギフトについて

SHOWROOM でも Instagram Live や TikTok ライブのようにギフトを送ることができます。

ギフトを送るには Show Gold というポイントを購入して送ります。
Show Gold はアプリで購入する場合 1G = 約1円、ブラウザで購入する場合 1G = 1円で、ブラウザで購入する方が少しお得です。

| 購入方法 | 価格 (1G あたり) |
| -------- | ---------------- |
| アプリ   | 約 1.1 円        |
| ブラウザ | 1 円             |

## タワーについて

10000 GOLD 以上のギフトまたは合計 10000 GOLD 以上の連投は「タワー」と呼ばれています。

タワーを送ると、メンバーからお礼がもらえます。
お礼の内容はメンバーによって異なります。
その日にお礼をする場合もあれば、別日にお礼配信としてまとめる場合もあります。
誕生日配信のときはメンバーによらずお礼配信としてまとめる傾向があります。

## 無料ギフトとミッションについて

SHOWROOM には無料ギフト (キラキラ星) があり、100 個まで贈ることができます。

またデイリー ミッションがあり、次の 3 つのミッションを完了して 30 GOLD もらうことができます。

- ギフトを贈る (無料ギフトでも OK)
- コメントを投稿する (最初に表示されているおすすめコメントでも OK)
- 配信を視聴する

デイリーミッションでもらえる GOLD は 2 日間の有効期限になっています。
そのため、次の流れがおすすめです。

1. 入室する。
2. おすすめコメントを投稿する。
3. 無料ギフトを贈る。
4. ミッションを完了する。(完了後 2, 3 分で GOLD が付与されます。)
5. 30 GOLD 分のギフトを贈る。
6. 配信を楽しむ。
