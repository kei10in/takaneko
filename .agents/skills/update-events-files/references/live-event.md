# Live Event

- **テンプレート -** `app/features/events/_event-template.ts` を使用する。
- `category` は `"LIVE"` とする。
- `region` は開催県名。海外の場合は都市名。
- タイムテーブル画像（`tags: ["timetable"]`）を追加・更新した場合は、読み取れる範囲で `acts.start` / `acts.end` に出演時間を反映する。時間が判別できない場合のみ既存の空値表現を維持する。
- ハッシュタグ #あしたのたかねこ のツイートは、リンクセクションに "#あしたのたかねこ" として追加する。

## セトリ

- セトリの情報源の優先順位は次の通り。
  - 星谷美来の X のポスト (https://x.com/Mikuru_hositani/)
  - 高音のなでしこ公式 X のポスト (https://x.com/takanenofficial/)
  - ファンによる X のポスト
- 星谷美来のポストを情報源にする場合、URL を `Act.url` に書く。
- 高音のなでしこ公式 X やファンによるポストの場合、URL を `Act.links` を書く。

## 本日のおチェキ

- たかねこまねーじゃーの X で「本日のおチェキ」ポストがされます。
- 本日のおチェキは画像としてイベントファイルの `meta.images` に追加します。
- リンクにも `[本日のおチェキ](URL)` の形式で、URL はチェキ投稿の URL を使います。
